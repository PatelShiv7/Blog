When creating my blog wireframes, I wanted it to be minimalistic while using colours that harmonize well with each other

It uses a simple layout for all devices while still displaying enough information for the user to understand what's going on.

The home page has a-links to all blog posts so that the user does not need to scroll down through each post. This was also used so that all the blog posts were sorted.

The blog has three navigation buttons at the top which are home, wellington zoo website, and the UXHub website.

Finally, it has a section where an image is displayed showing the code or a screenshot of the result.

All annotations are on the figma link.
