Learning about the history of JavaScript has made me think about how much JavaScript has developed over the past 20 years. Application Programming Interface (API) is an interface between two implementations. API’s provide a set of operations which is defined by inputs and outputs; and second, it allows reimplementation without compromising its users (Sandoval, 2018). API’s were invented by Maurice Wikes, David Wheeler, and Stanley Gill (Sandoval, 2018).

API’s are used in many types of websites such as commerce, social media, cloud computing, and mobile. Without API’s, websites such as Facebook wouldn’t allow users to access their friends, photos, events and profile information. Everything is become more mobile. Google maps was invented in 2006 in which the Google Maps API was made using reverse engineering and “hacking the JavaScript application”.

“Location became a major topic of the API conversation that we were about to have on every mobile device solid in the next couple of years” (Lane, 2019). JavaScript has developed over the past 20 years, and is still “only just beginning the journey”. Web APIS can be used to connect almost to the internet, and are “regularly being used to invent entirely new products and services, as well as constructing ecosystems” (Lane, 2019).
