For my first website, I wanted to recreate the Wellington Zoo webpage. This used images from their images folder which linked to external websites on their page.

At the top of the page, it has an interactive javascript slideshow where the user can cyle three images relating to the zoo. I've also added a navigation bar which has an a:hover feature in the CSS.

For the rest of the website, I've added images which mostly have a-links attached to them which link to external websites

I've also implemented a search bar which allows the user to enter their emails for newsletter updates by using a form.
