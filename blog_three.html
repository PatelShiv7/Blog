For this tutorial, the task was to create a flexbox which had multple class names in one class wrapper named flex-wrapper which holds all the items inside.

The flex wrapper has a width of 57 vw which maintains this width through different screen sizes with a flex display allowing the user to adjust where content is defined on the page

A media query is used for mobile devices with a max width of 600px which readjusts the width and changes the flex direction from row to column as it now has a smaller screen size.

Other components have also been changed such as the font-size, the background colour and other screen dimensions.
