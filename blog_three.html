It's important that designers think about the psychological aspects in a website to focus on what users expect on one. I never really thought about the psychology side that much until this lecture which was quite good. There are many techniques to apply psychology to websites such as the serial position effect which is subcategorised into the primary effect and the recency effect.

The primary effect states that users tend to recall first items with greater accuracy due to the small amount of processing effort spent rehearsing them. The serial position effect makes it so that you can control user behaviour by placing items or information in a particular order. This idea is mostly used in e-commerce websites where they list the expensive item first then the cheapier onces. This would encourage the user to make the purchase on the expensive item.
