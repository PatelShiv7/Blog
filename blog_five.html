Javascript can be a very powerful language to learn when creating interactivity onto a website.

For this tutorial, our goal was to use javascript to read the DOM of the html files and change elements depending on what the user clicks.

For mine, I wanted to use javascript to change from either a sun or a moon. I created a button using html and created a JS file which read this button. Depending on what the user clicks, will change the image on the screen.
