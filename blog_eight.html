JavaScript is better know as The Modern Web. There are many other type of JavaScript frameworks such as Backbonejs, knockoutjs, angular, react, javascript, es6, typescript, and babel. This is just a list of a few and there are so many more languages that have been developed over the past 20 years.
The modern web consists of 4 characteristics which are HTML / CSS / JS, Jquery, Frameworks and Tooling. HTML is the basic structure of a site which is enhanced by other technologies such as CSS and JavaScript. jQuery is designed to simplify the client-side scripting of HTML such as interactive slideshows.
Javascript consists of many framework libaries that consist of pre-written JavaScript such as

Angular - Powerful, efficient, and open-source JavaScript framework which allows the coder to extend the HTML inside the application and "interprets the attrivutes to perform data binding" (Arora, 2021)

React - Used to develop and operate the dynamic user itnerface of the web page with high incoming traffic. Allows for a virtual DOM. Websites include code academy, facebook, instagram and yahoo mail.

JSX - allows the coder to write HTML in react - makes life easier.
