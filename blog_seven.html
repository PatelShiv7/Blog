For our group project, we decided to use Cody's pitch which was a wiki page aimed towards user experience content.

This website would have five pages which are the main page, articles, resources, videos, and submit content.

Most of these pages will have a similar layout to keep the websites consistency. Some of these pages will also have a filter section where the user can filter out content based on their needs We needed to create a style guide which will be used for our website. We've mainly used the colours orange, black and white as they harmonize well with each other.

Headings will be using IBM Plex Sans & paragraphs will use Open Sans. These are easy to read fonts which will be useful for all users, especially those with reading disabilities.
