CSS Grid is a better layour system than flexbox as it's two dimensional compared to flexbox's which are mainly 1D.

Grid's can define container elements such as display: grid, setting the column and row sizes with grid-template-columns and grid-template-rows, then placing child elements inside the grid using grid-column and grid-row.

CSS can place these in any order, which makes it easy to rearrange the grid with media queries.

CSS Grids are also good with view width (vw) & view height (vh) dimensions because most aspects on your website can be responsive without needing to use media queries for items such as hero shots, footers, etc.
